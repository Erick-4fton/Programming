# Dev Archive - GitHub Pages

Website ini dideploy menggunakan GitHub Pages.

## Cara Update Website:
1.  Tambahkan konten baru di folder `content/` dalam format Markdown.
2.  Jalankan skrip konversi (jika sudah lengkap): `python tools/convert.py`
3.  Commit dan Push perubahan ke GitHub.

## Struktur Folder:
- `/assets`: File CSS, JS, dan Gambar.
- `/content`: Sumber tulisan (Markdown).
- `/writeups`: Artikel yang sudah jadi (HTML).
- `/tools`: Skrip otomasi.
